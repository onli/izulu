izulu

izulu is a free script which changes the background of your GNU/Linux-Desktop according to the weather.

Features

  • React via weather APIs to the weather, sunset and sunrise.
  • Distinguish between 2x14 weather states in 5 categories (day -> rain -> light rain).
  • Show temperature and forecast.
